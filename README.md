# Usage
## For Linux user
Simply compile the code yourself on your Linux OS machine using gfortran.
gfortran fort.f90 -o fort
f95 fort.f90 -o fort

## For Window user
Install g++ gcc gfortran, then install fortran IDE Program, says Geany text editor. https://www.geany.org/
